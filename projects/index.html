Featured Project 01

Sustainable Marin - Website Migration

I migrated Sustainable Marin from a custom build to WordPress so their team can update pages faster without relying on a developer for routine changes.

Challenge

Their previous site made basic edits slow because most updates required developer support.

Solution

I rebuilt the site in WordPress with reusable page patterns, clearer structure, and an editing workflow staff can use directly.

Outcome

I gave the team a self-serve CMS, faster publishing, and a lower-maintenance site with fewer routine dev requests.

  • I moved page updates from developer-needed to staff self-serve.
  • I standardized layouts in WordPress so edits stay consistent across pages.
  • I reduced long-term maintenance overhead for routine site updates.

Featured Project 02

MarqueeKit - Reusable HTML/CSS/JS Marquee System

I built MarqueeKit after struggling to find solid marquee examples for my own sites. It is a repeatable HTML/CSS/JS setup I can implement across projects.

Challenge

I could not find marquee examples that were smooth, easy to customize, and reliable across different sites.

Solution

I built a custom HTML/CSS/JS marquee pattern with repeatable structure, responsive behavior, and straightforward setup for new pages.

Outcome

Anyone can now go to marqueekit.com, purchase the solution, and implement it in their own projects instead of rebuilding marquee logic from scratch.

  • Built as a repeatable HTML/CSS/JS solution, not a one-off snippet.
  • Available as a purchasable product at marqueekit.com for direct project use.
  • Ongoing updates include performance tuning and maintenance fixes.
"Setup was clear, and the motion was smoother than the marquee code we used before."
